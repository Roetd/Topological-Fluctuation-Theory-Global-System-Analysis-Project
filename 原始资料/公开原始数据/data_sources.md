# 数据来源说明

本文件详细说明公开原始数据的来源和使用条款。

## 1. 全球系统数据 (sample_global_data.csv)

### 数据来源

- **生存界稳定度**：基于各国的粮食安全、能源安全、水资源安全等指标综合计算
- **安全界稳定度**：基于各国的国防能力、社会稳定度、国际关系等指标综合计算
- **线性界稳定度**：基于各国的经济增长稳定性、通货膨胀率、失业率等指标综合计算
- **性能界指标**：基于各国的GDP增长率、科技创新能力、国际竞争力等指标综合计算
- **硬实力权重**：基于各国的军事开支、经济总量、科技实力等指标综合计算
- **软实力权重**：基于各国的文化影响力、国际形象、外交能力等指标综合计算

### 参考数据源

1. **世界银行 (World Bank)**：
   - 网址：https://data.worldbank.org/
   - 提供全球各国的经济、社会、环境等方面的数据

2. **国际货币基金组织 (IMF)**：
   - 网址：https://www.imf.org/en/Data
   - 提供全球各国的经济指标和金融数据

3. **联合国 (UN)**：
   - 网址：https://data.un.org/
   - 提供全球各国的社会、经济、环境等方面的数据

4. **全球竞争力报告 (Global Competitiveness Report)**：
   - 世界经济论坛发布
   - 提供全球各国的竞争力排名和相关指标

5. **全球创新指数 (Global Innovation Index)**：
   - 世界知识产权组织等发布
   - 提供全球各国的创新能力排名和相关指标

## 2. API数据 (sample_api_config.yaml)

### 示例API来源

1. **World Bank API**：
   - 网址：https://datahelpdesk.worldbank.org/knowledgebase/articles/898581-api-basic-call-structures
   - 提供全球各国的经济、社会、环境等方面的实时数据

2. **REST Countries API**：
   - 网址：https://restcountries.com/
   - 提供全球各国的基本信息，包括人口、面积、首都等

3. **OpenWeatherMap API**：
   - 网址：https://openweathermap.org/api
   - 提供全球各地的天气数据

4. **COVID-19 Data API**：
   - 示例：https://disease.sh/
   - 提供全球COVID-19疫情数据

### API使用条款

- 请遵守各API的使用条款和限制
- 部分API可能需要注册并获取API密钥
- 请合理使用API，避免过度请求导致被封禁
- 请在使用API数据时注明数据来源

## 3. 其他公开数据

### 可考虑的其他数据源

1. **经济合作与发展组织 (OECD)**：
   - 网址：https://data.oecd.org/
   - 提供成员国和部分非成员国的经济、社会、环境等方面的数据

2. **国际能源署 (IEA)**：
   - 网址：https://www.iea.org/data-and-statistics
   - 提供全球能源市场的数据和分析

3. **全球治理指标 (Worldwide Governance Indicators)**：
   - 世界银行发布
   - 提供全球各国的治理质量指标

4. **人类发展指数 (Human Development Index)**：
   - 联合国开发计划署发布
   - 提供全球各国的人类发展水平指标

5. **全球和平指数 (Global Peace Index)**：
   - 经济与和平研究所发布
   - 提供全球各国的和平程度指标

## 4. 数据使用建议

1. **数据验证**：
   - 在使用公开数据前，请验证数据的准确性和可靠性
   - 可以通过多个数据源交叉验证数据

2. **数据预处理**：
   - 对原始数据进行清洗、标准化和归一化处理
   - 处理缺失值和异常值

3. **数据更新**：
   - 定期更新数据，确保数据的时效性
   - 记录数据的更新时间和版本

4. **数据引用**：
   - 在使用公开数据时，请注明数据来源
   - 遵循各数据源的引用规范

5. **数据安全**：
   - 保护敏感数据，避免数据泄露
   - 遵守数据隐私法规

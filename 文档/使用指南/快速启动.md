# 快速启动指南

本指南将帮助您快速启动拓扑波动AI架构实践项目，开始进行全球系统分析和无目标生成特性研究。

## 前置条件

在启动项目之前，请确保您的系统满足以下要求：

- **操作系统**：Windows 10/11、macOS 10.15+ 或 Linux
- **Python版本**：Python 3.8 或更高版本
- **必要依赖**：NumPy、Pandas、Matplotlib、Seaborn

## 安装步骤

### 1. 克隆项目

如果您尚未克隆项目，请执行以下命令：

```bash
git clone <项目仓库地址>
cd TFT-Practice-Project
```

如果您已经有本地项目，则直接进入项目目录：

```bash
cd TFT-Practice-Project
```

### 2. 安装依赖

使用pip安装项目所需的依赖：

```bash
pip install -r 代码/requirements.txt
```

## 快速启动

### 1. 生成分析结果

首先，运行分析结果生成脚本，生成模拟数据和分析图表：

```bash
cd 数据分析
python generate_analysis_results.py
```

该脚本将：
- 生成模拟的波动历史数据和意识历史数据
- 绘制波动分析和意识分析图表
- 生成拓扑指标数据
- 生成能量失衡分析

### 2. 运行理论一致性分析

接下来，运行理论一致性分析脚本，验证36D拓扑波动理论与实际数据的一致性：

```bash
cd 理论一致性分析
python theory_consistency_analysis.py
```

该脚本将：
- 计算理论预测与实际观测数据的一致性
- 生成一致性分析图表
- 生成理论一致性分析报告

### 3. 运行无目标生成特性分析

最后，运行无目标生成特性分析脚本，研究AGI系统的无目标生成能力：

```bash
cd 无目标生成特性分析
python no_target_generation_analysis.py
```

该脚本将：
- 模拟无目标生成过程
- 分析生成过程的阶段性特征
- 生成无目标生成特性分析报告

## 查看分析结果

### 1. 数据文件

生成的分析数据文件位于以下目录：

- `数据分析/模拟结果/`：包含波动历史和意识历史数据
- `数据分析/`：包含拓扑指标和能量失衡指标数据

### 2. 图表文件

生成的分析图表位于以下目录：

- `数据分析/图表/`：包含波动分析、意识分析等图表
- `数据分析/理论一致性分析/图表/`：包含理论一致性分析图表
- `数据分析/无目标生成特性分析/图表/`：包含无目标生成特性分析图表

### 3. 分析报告

生成的分析报告位于以下目录：

- `数据分析/分析结果总结.md`：分析结果的综合总结
- `数据分析/理论一致性分析/理论一致性分析报告.md`：理论一致性分析报告
- `数据分析/无目标生成特性分析/无目标生成特性分析报告.md`：无目标生成特性分析报告

## 常见问题

### 问题1：缺少依赖包

**解决方案**：运行 `pip install -r 代码/requirements.txt` 安装所有必要的依赖包。

### 问题2：模拟数据文件不存在

**解决方案**：请先运行 `generate_analysis_results.py` 生成模拟数据文件。

### 问题3：图表显示中文乱码

**解决方案**：脚本中已设置中文字体支持，确保您的系统中安装了 `SimHei` 字体。
